## Overview
A consolidated home server environment hosting personal services, experiments, and dashboards. Emphasis on maintainability, backups, and low-friction access.

## Highlights
- Reverse proxy and TLS via Nginx Proxy Manager (NPM)
- Containerized services (media, monitoring, utilities)
- Automated backups and update cadence
- Network segmentation where practical

## Notes
This is an infrastructure/ops project without a public repository.
