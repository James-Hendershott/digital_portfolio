## Overview

A suite of tools I built to manage vendor relationships and fleet maintenance at my current job as a Vendor Service Manager at Amerit Fleet Solutions. What started as a simple email utility evolved into two full production applications — a 20,000+ line JavaScript dashboard I use every day at work, and a MERN stack rebuild designed for team-wide deployment. The total codebase across both repos exceeds 31,000 lines of application code and 10,000 lines of documentation.

## Problem

As a Vendor Service Manager overseeing fleet maintenance across 6 Target DC locations (West Jefferson OH, Upper Marlboro MD, Lake City FL, Denton TX, Waterloo IA), I was spending excessive time on:
- Repetitive vendor emails with copy/paste across 40+ contacts
- Manual data entry and tracking across a 40-column Smartsheet
- No visibility into trends, aging units, or cost patterns
- Difficulty coordinating with team members on unit status
- No way to quickly detect units returned to service or past-due estimates

## Solution

**Phase 1 — Dashboard (2024–Present, daily production use):** Built a zero-dependency browser tool that connects directly to Smartsheet, providing real-time dashboards, 3 report types, 9 workflow forms, and templated vendor communications. Iterated daily based on real workflow pain points over 2 years. Every vendor interaction at work flows through this tool.

**Phase 2 — MERN Rebuild (2026–Present):** Rebuilding as a proper full-stack application with its own database, enabling offline capability, better performance, team collaboration, and company-wide rollout. Backend is substantially complete; frontend needs 7 more workflow forms.

## Architecture Highlights

**Dual-Write Pattern:** Changes write to MongoDB immediately for fast UI response, then BullMQ queues Smartsheet sync jobs (5 types) for eventual consistency. Jobs retry 3x with exponential backoff starting at 2 seconds. Failed syncs update `syncStatus` on the document.

**Tiered Caching:** Redis caches at different TTLs based on data volatility — dashboard summaries every 2 minutes, unit data every 5 minutes, contacts every 10 minutes, locations every 30 minutes. Mutations invalidate related cache keys via pattern matching.

**Real-time Updates:** Socket.io broadcasts `unit:created` and `unit:updated` events to all clients. React Query caches auto-invalidate on receipt, so team members see live status without refreshing.

**AI-Assisted Workflows:** Claude Sonnet analyzes vendor estimate documents (PDFs, images) with two-pass extraction and price comparison for auto-tagging (Basic vs OEM). Claude Haiku provides 4-mode text improvement with form-specific context awareness. AI also rewrites internal comments into customer-facing language for reports.

**CORS Proxy Failover:** The web dashboard routes Smartsheet API calls through a CORS proxy array with automatic failover, 2 retries per proxy, and exponential backoff — making it resilient to proxy outages.
