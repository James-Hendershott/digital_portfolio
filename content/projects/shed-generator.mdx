## Overview

A comprehensive, interactive guide for planning and building a gravel pad foundation for a backyard shed in Farmington, UT. Includes everything from permits to material calculations to step-by-step construction instructions.

## Features

- **Material Calculator**: Automatic calculations for gravel, landscape fabric, and edging
- **Cost Estimates**: Real-time cost breakdowns based on local supplier pricing
- **Step-by-Step Guide**: Detailed instructions with tips for each phase
- **Local Resources**: Links to nearby suppliers and rental equipment
- **Mobile-Friendly**: Responsive design for on-site reference

## Tech Stack

Single-page HTML application with:
- CSS custom properties for theming
- Vanilla JavaScript for calculations
- No build process or dependencies
- Dark mode UI
